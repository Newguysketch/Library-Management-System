Library Management System

Features:
- Add, Update and Remove Books
- Register and Update Members
- Borrow and Return Books
- Due Date Tracking
- Borrowing History
- Current Loans
- Premium Membership Support
- Input Validation
- Error Handling

How to Run:

1. Open terminal in src folder
2. Compile:
   javac *.java

3. Run:
   java LibraryManagementSystem
